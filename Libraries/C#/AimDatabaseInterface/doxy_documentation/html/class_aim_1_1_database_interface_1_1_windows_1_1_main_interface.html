Detailed Description

The main interface.

This control contains the following:

  • A SearchControl to allow the user to select an item to modify.
  • An IEditor to provide the actual functionality to create/modify data.
  • A SearchControl to allow the user to select a data for controls such as ListboxSelectorEditorControl.
  • A delete button that should 'just work'.
  • A create button that should 'just work'.
  • An apply/save button that should 'just work'.
  • The ability to resize parts of the interface, to allow the user to customise what they're seeing.
  • A status bar.
