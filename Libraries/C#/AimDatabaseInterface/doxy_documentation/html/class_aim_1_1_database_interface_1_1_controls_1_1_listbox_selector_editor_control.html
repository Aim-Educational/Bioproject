Detailed Description

Similar to the ListboxEditorControl in that it contains a list of items the user can mess around with, but with some unique features.

The list can be reordered using buttons.

The values have to be searched with similar to how SelectorEditorControl functions.

Please make sure ListboxSelectorEditorControl.provider and ListboxSelectorEditorControl.mainInterface are set before this control is used by the user.
